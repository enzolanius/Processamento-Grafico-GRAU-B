Descrição do Projeto

Este projeto é um jogo desenvolvido em C++ utilizando OpenGL, GLFW e GLAD, com visualização em projeção isométrica (Diamond Isometric).

O objetivo do jogador é explorar o mapa, coletar todas as moedas espalhadas pelo cenário e alcançar a saída final, evitando perigos como lava e obstáculos bloqueantes.

O projeto foi desenvolvido para a disciplina de Processamento Gráfico – Grau B.

Objetivos do Jogo

O jogador deve:

Percorrer o mapa isométrico.
Coletar todas as moedas disponíveis.
Evitar perder vidas ao caminhar sobre lava.
Encontrar a saída do mapa.
Entrar na saída somente após coletar todas as moedas.
